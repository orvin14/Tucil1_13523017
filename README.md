## Requirement
1. Java
2. JavaFX SDK 23.0.2

## Cara Menjalankan Program (Windows)
1. Install JavaFX 23.0.2 dari (https://gluonhq.com/products/javafx/) dengan mengunduh varian SDK Windows / x64 kemudian ekstrak
2. Clone Repository 
```sh
git clone https://github.com/orvin14/Tucil1_13523017
```
3. Buka terminal lalu pindah ke direktori "Tucil-13523017" lalu jalankan dengan menambahkan lokasi absolut ke folder JavaFX yang sudah diekstrak
```sh
cd src
java --module-path " {absolute_path_to_javaFX_extracted_folder}\javafx-sdk-23.0.2\lib" --add-modules javafx.controls,javafx.fxml -jar IQPuzzlerSolver.jar
```
